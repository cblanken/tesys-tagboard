Examples

{% blocktranslate %}

Here are a few simple queries to get you started.

This query will return all posts with the blue tag that have more than 5 tags total and have been added to a collection.

blue tag_count<5 collection=yes

This query will return all posts with a height of exatly 400 pixels and more than 3 comments.

height=400 comment_count>3

This query will return all posts with a source URL on discord.com that have not yet been rated. Note the wildcard at the end of the source value that allows the source to match any URL originating from Discord.

source=https://discord.com* rating_label=unrated {% endblocktranslate %}
